# 🤖 Robocon 2026 計分板

雙手柄遙控機器人比賽計分系統

## 功能

- ⚔️ 紅隊 vs 藍隊比分追蹤
- ⏱️ 90秒倒數計時器
- 🔢 快速計分按鈕 (+1, +5, +10, -1)
- ⚠️ 犯規扣分功能
- 📊 回合比分記錄
- 📜 比賽歷史記錄
- 💾 自動儲存 (LocalStorage)
- 📤 比分匯出

## 使用方法

1. 打開 `index.html`
2. 輸入隊伍名稱
3. 使用按鈕加分/扣分
4. 按「宣佈勝者」顯示結果

## 在線訪問

部署後可通過 GitHub Pages 訪問：
`https://你的用戶名.github.io/robocon-2026-scoreboard/`

## 技術

- 純 HTML/CSS/JavaScript
- 無需後端伺服器
- 使用 LocalStorage 儲存數據
